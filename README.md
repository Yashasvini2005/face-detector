# Face Detector

Real-time face detection using OpenCV and Python.

## Features
- Detects faces from webcam feed
- Green rectangle around each face
- Press 'q' to quit
- Press 's' to save screenshot

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run: `python detect.py`
